# Security Policy

## Supported Versions

Security fixes are considered for the latest published `0.x` release.

## Reporting a Vulnerability

Please report security issues privately by opening a GitHub security advisory or
by contacting the package maintainers through the repository's listed security
contact.

Do not publish exploit details until a fix or mitigation is available.

## Local Data

Clovis stores financial records in a local SQLite database. Keep backups, protect
ledger files with normal operating-system permissions, and review imported data
before posting it.
