# Clovis Changelog

This page is the human-facing release index for Clovis.

The canonical version-by-version changelog lives in [`../CHANGELOG.md`](../CHANGELOG.md).
GitHub Releases are the canonical published release records:

- [Latest GitHub Releases](https://github.com/cloviscomputing/clovis/releases)
- [npm package](https://www.npmjs.com/package/clovis)

Release notes should explain behavior changes in plain language, especially
changes to SQLite schema, MCP tools, CLI behavior, import safety, report
semantics, and package entrypoints.

For the publishing procedure, see [`../RELEASING.md`](../RELEASING.md).
